## 운영 규칙
- 기능 추가/삭제/승격 요청은 본 문서 등록 전 개발 착수 금지.
- PM/Tech Lead 승인 전 상태는 `Proposed`로 유지.

- 반려 항목은 `Rejected`로 종료하고 근거를 남긴다.

## 상태 정의
- `Proposed`: 요청 등록
- `Reviewing`: 영향도/일정/품질 검토 중
- `Approved`: 승인 완료, 반영 가능
- `Rejected`: 반려
- `Shipped`: 릴리스 반영 완료
