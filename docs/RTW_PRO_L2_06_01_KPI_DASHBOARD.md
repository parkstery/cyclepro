## 핵심 KPI
- 로그인 성공률
- 룸 입장 성공률
- 20분 완주율
- 평균 동기화 지연
- 동기화 지연 P95/P99
- 추월 이벤트 반영 정확도
- 센서 재연결 성공률
- 센서 인증 주행 비율(Verified Ride Ratio)
- 7일 재참여율

## KPI 세그먼트
- 센서 보유 그룹 / 센서 미보유 그룹을 분리해 집계한다.
- 네트워크 상태(양호/불안정) 구간별로 지연/완주율을 분리한다.
- 신규 사용자(7일 이내) / 기존 사용자(8일 이상) 재참여율을 분리한다.

## 규칙
- 주간 단위로 추세를 확인한다.
- 목표 미달 지표는 원인/액션 아이템을 기록한다.
- 평균값만으로 판단하지 않고 P95 이상 구간을 함께 확인한다.
- Go/No-Go 판정 입력 스냅샷(`BetaKpiSnapshot`)과 동일 정의를 사용한다.
- 주간 보고 포맷은 `L2-06-05 Weekly Beta Report Template`을 사용한다.
