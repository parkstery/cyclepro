## 목적
- 지도 초기화, 경로 표시, 라이더 위치 표시의 기준 동작을 정의한다.

## 핵심 규칙
- 지도 첫 렌더 실패 시 재시도한다.
- 경로 폴리라인은 OSRM 응답 좌표를 그대로 사용한다.
- 현재 위치 마커는 보간값 기준으로 갱신한다.
- 첫 렌더 오케스트레이션은 `MapFirstPaintCoordinator`에서 관리한다.
- 사용자 실패 안내는 재시도 소진 시 단일 메시지로 노출한다.

## QA 체크리스트
- [ ] 첫 렌더 시간 측정
- [ ] 폴리라인 누락 없는지 확인
- [ ] 마커 갱신 끊김 확인
- [ ] 렌더 실패 시 재시도 횟수/지연 정책 일치 확인
- [ ] 재시도 소진 시 사용자 오류 메시지 확인
