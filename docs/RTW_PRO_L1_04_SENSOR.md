## 우선 지원 범위
- BLE Cycling Speed and Cadence(CSC) 1종 우선
- Android 10+ 기준 권한/스캔 정책 준수

## UX 플로우
1. 센서 검색 시작
2. 기기 선택 및 연결
3. 실시간 값 수신(속도/케이던스)
4. 끊김 시 자동 재시도(최대 N회)
5. 실패 시 가상 입력 모드 전환 제안

## 데이터 보정 규칙
- 스파이크 필터: 순간 속도 급변 임계치 초과 시 완화
- 이동평균 스무딩: 최근 N샘플 평균 사용
- 드롭아웃 처리: 일정 시간 무수신 시 속도 0으로 감쇠
- 최대 속도 상한: 비정상 값 차단

## 센서 미보유 모드
- 슬라이더 또는 프리셋 속도 입력 제공
- 동일한 레이스 파이프라인 사용(센서만 대체)

## 완료 기준
- 센서 연결/재연결/실패 fallback 흐름이 문서화된다.
- 센서 이벤트 -> 속도 계산 -> 서버 전송 경로가 명확히 정의된다.
