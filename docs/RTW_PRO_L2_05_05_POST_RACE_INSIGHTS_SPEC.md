## 목적
- 주행 후 화면에서 Pro 사용자에게 필요한 최소 분석 인사이트를 제공한다.

## 필수 인사이트
- 추월 타임라인: 내가 추월/피추월된 시점을 시간축으로 표시
- 구간 페이스 편차: 코스를 4~8개 구간으로 나누어 평균 속도 편차 표시
- 피니시 분석: 상위권 대비 마지막 20% 구간의 속도 차이
- PB 비교: 동일 코스 개인 최고기록 대비 개선/악화 구간

## 표시 규칙
- 첫 화면은 3초 내 이해 가능한 요약 카드 2~3개만 노출한다.
- 상세 분석은 펼침(드릴다운) 구조로 제공한다.
- 수치 해석을 돕기 위해 "좋음/보통/개선 필요" 라벨을 함께 표기한다.

## 액션 유도
- 즉시 재도전 버튼
- 다음 정기 이벤트 알림 설정
- 결과 공유 카드 생성(외부 공유는 후순위 가능)

## 완료 기준
- 사용자가 결과 화면에서 다음 행동(재도전/알림설정) 중 하나를 선택할 수 있다.
- 사용성 테스트에서 인사이트 이해도 평균 4.0/5.0 이상을 달성한다.
