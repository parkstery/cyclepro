## 목적
- 경쟁 경험의 신뢰도를 정량화하기 위한 정확도 지표를 정의한다.

## 측정 항목
- 순위 일관성: 서버 최종 순위와 클라이언트 표시 순위의 불일치율
- 추월 이벤트 정확도: 실제 순위 역전 시점 대비 UI 이벤트 반영 오차
- 갭 표시 정확도: 서버 기준 거리 차이와 HUD 갭 값의 평균 오차
- 지연 분포: 평균 지연, P95, P99

## 목표 기준
- 순위 불일치율 1% 이하
- 추월 이벤트 반영 정확도 95% 이상
- 갭 표시 평균 오차 5m 이하(근접 구간 기준)
- 동기화 지연 평균 1.5초 이하, P95 2.5초 이하

## 수집/판정 규칙
- 레이스 로그는 서버 타임스탬프를 기준으로 오프라인 재계산한다.
- 클라이언트 표시는 1초 간격 샘플링 로그로 비교한다.
- 사용자 체감과 수치가 다를 때는 UI 보간/애니메이션 지연을 분리 분석한다.

## 완료 기준
- KPI 대시보드와 Go/No-Go 문서에서 동일 용어/임계치로 참조된다.
- 베타 기간 동안 주간 리포트 템플릿에 본 지표가 포함된다.
